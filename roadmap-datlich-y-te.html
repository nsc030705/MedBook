🛠 Tech Stack Đề Xuất
Frontend + Backend
Next.js 14
App Router + API Routes + Server Components. Một repo duy nhất, dễ deploy.
Language
TypeScript
Bắt lỗi sớm, code rõ ràng hơn, hội đồng đánh giá cao.
UI / Styling
Tailwind + shadcn/ui
Nhanh, đẹp, responsive sẵn, không cần thiết kế từ đầu.
ORM
Prisma
Migration tự động từ schema, type-safe query, tài liệu tốt.
Database
PostgreSQL
Hỗ trợ JSONB, full-text search, pgvector cho RAG — dùng một DB duy nhất.
Auth
NextAuth.js v5
OAuth Google miễn phí, JWT session, không tự code auth.
AI Chatbot
OpenAI API / Gemini
Tool calling để truy vấn lịch rảnh DB. Free tier đủ cho đề án.
RAG / Vector
pgvector + LangChain
Dùng luôn PostgreSQL làm vector store — không cần DB riêng cho đề án.
Email
Resend
Free 3000 email/tháng, API đơn giản, tích hợp Next.js dễ.
Deploy DB
Supabase (free)
PostgreSQL hosted miễn phí, pgvector extension sẵn, dashboard trực quan.
Deploy App
Vercel (free)
Zero config với Next.js, HTTPS tự động, URL đẹp cho demo.

Cuối tuần 10
Nộp & bảo vệ đề án
Báo cáo hoàn chỉnh, slide demo sẵn sàng, production deploy ổn định, bảo vệ thành công.
⚠️ Đừng để phần AI đến tuần 8 mới làm. AI chatbot là điểm nhấn của đề án nhưng cũng dễ bị tắc nhất — tích hợp LLM API, xử lý tool calling, debug RAG đều tốn thời gian hơn dự kiến. Bắt đầu từ tuần 6 để còn thời gian sửa nếu có vấn đề.
⚠️ Không cần RAG phức tạp. Với mức độ đề án, chỉ cần 10–20 tài liệu y tế nhỏ, embed bằng pgvector, truy xuất top-3 chunk gần nhất là đủ để demo ấn tượng. Không cần hybrid search hay reranker.
💡 Mẹo cho hội đồng: Hội đồng thường ấn tượng với tính năng AI chatbot kiểm tra lịch rảnh hơn là UI đẹp. Demo kịch bản: người dùng hỏi bằng tiếng tự nhiên "Bác sĩ Nguyễn còn lịch thứ 4 tuần này không?" và chatbot trả lời chính xác — đây là phần phân biệt đề án của bạn với những đề án CRUD thông thường.
🚀 5 Bước Đầu Tiên Cần Làm Ngay
1
Khởi tạo project
npx create-next-app@latest
Chọn TypeScript + Tailwind + App Router.
Cài thêm: prisma, shadcn/ui, next-auth
2
Setup PostgreSQL + Prisma
Tạo DB trên Supabase (miễn phí).
Enable extension pgvector.
Viết schema.prisma — 6 bảng cốt lõi trước.
Chạy npx prisma migrate dev
3
Setup Auth
Cài next-auth@beta + Google Provider.
Kết nối với bảng users + sessions.
Test login/logout hoạt động trên Vercel.
4
Dựng trang đặt lịch
Trang tìm bác sĩ có filter chuyên khoa.
API: GET lịch rảnh + POST tạo appointment.
Gửi email xác nhận với Resend.
5
Tích hợp AI chatbot
npm install openai langchain
Định nghĩa tool check_doctor_schedule.
Stream response về giao diện chat widget.
